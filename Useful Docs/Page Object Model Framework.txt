Page Object Model Framework

Rules
1. Every page should be a separate class
2. There should be a reusable methods class 
3. Every page should extend that reusable method class
4. Verify the page it is landed
5. Every action should be a separate method
6. Every method should have a return type and return to the page it lands post completion of that action


Steps
1. Create my project --> Maven
2. Create my resusable method class (BaseClass or WrapperClass) in src/main/java
3. Create my page (Every page should be a separate class, extends reusable methods class,)
4. Create a constructor in each page and verify a unique element of that page is displayed
5. Create a separate method for each action in the respective page
6. Every method should have a return type and return to the page it lands post completion of that action
7. Create your test cases and execute it
